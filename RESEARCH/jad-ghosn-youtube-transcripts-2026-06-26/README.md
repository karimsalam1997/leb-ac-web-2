# Jad Ghosn YouTube Transcripts, 2026-06-26

This folder contains the ten newest transcript/subtitle downloads I found from the live Jad Ghosn YouTube channel on 2026-06-26.

I searched for exact `Jad Rassoun` / `Rassoun` matches locally and on YouTube-style web results, but the matching public channel and existing local material point to Jad Ghosn / جاد غصن.

## What is here

- `vtt/`: raw YouTube Arabic original auto-caption files, with timestamps.
- `transcripts_txt/`: readable cleaned text versions.
- `metadata.csv`: the list of the ten newly saved videos.
- `already_saved_before_this_run.csv`: the Jad Ghosn / هامش جاد transcripts that were already saved in the June 24 research pack before this run.
- `metadata_json/`: small metadata snapshots for each newly saved video.

## Caution

These are YouTube automatic captions, not hand-corrected transcripts. They are useful for search and first-pass reading, but names, places, and political terms should be checked against the video before quoting.
