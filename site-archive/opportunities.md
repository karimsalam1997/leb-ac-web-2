# Lebanese Academic Opportunity Report

This is the running list of strong ideas discovered during automation cycles but not automatically implemented.

The automation may add to this file while working. It should not treat every idea here as permission to build it immediately.

## Starting Opportunities

- Build a stronger Issue 01 cover experience, not just an essays index.
- Modernize topic filters so they feel like an editorial instrument rather than old category buttons.
- Explore a restrained custom cursor or delayed cursor marker that feels literary and precise.
- Improve essay reading pages with better rhythm, progress, pull quotes, image pauses, and mobile pacing.
- Create a clearer visual grammar for article images, issue images, and archival images.
- Audit longform prose for AI-like rhythm, generic abstraction, over-balanced transitions, and placeless paragraphs.
- Add subtle interaction: hover motion, image reveals, reading progress, animated topic filtering, and section transitions.
